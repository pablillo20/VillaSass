    mixins

    genera-cabeceras

    @mixin genera-cabeceras($level, $color) { ... }

    Description

    Mixin para generar cabeceras (h1 a h6) con diferentes tamaños y colores.

    Este mixin se encarga de aplicar el tamaño de fuente y color según el nivel de la cabecera (h1 a h6), utilizando un esquema de colores y tamaños predefinidos.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $level

    El nivel de la cabecera (1 a 6).

    		Number— none
    $color

    El color de la cabecera (puede ser uno de los valores definidos como 'primary', 'secondary', etc.).

    		Color— none

    Requires

    • [function] size
    • [function] color
    • [variable] font-family

    flex

    @mixin flex($direction, $justify, $align) { ... }

    Description

    Mixin para crear un contenedor flex.

    Este mixin crea un contenedor con el modelo Flexbox para alinear y distribuir los elementos de acuerdo con las opciones proporcionadas (dirección, justificación y alineación).

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $direction

    La dirección del eje principal (por defecto es row). Puede ser row o column.

    		String— none
    $justify

    Cómo justificar los elementos a lo largo del eje principal (por defecto es center).

    		String— none
    $align

    Cómo alinear los elementos a lo largo del eje transversal (por defecto es center).

    		String— none

    media

    @mixin media($size) { ... }

    Description

    Mixin para aplicar estilos específicos para diferentes tamaños de pantalla (media queries).

    Utiliza el mixin media para envolver contenido dentro de una consulta de medios que se activará cuando el tamaño de la pantalla sea menor o igual a un determinado breakpoint.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    El tamaño de la pantalla (puede ser uno de los valores definidos como 'small', 'medium', etc.).

    		String— none

    Requires

    • [function] breakpoint

    fafaIcons

    @mixin fafaIcons($color, $font-size) { ... }

    Description

    Mixin para agregar estilos a los íconos de FontAwesome.

    Este mixin aplica estilos personalizados a los íconos de FontAwesome, incluyendo el tamaño de la fuente y el color, permitiendo que sean fácilmente reutilizados en toda la aplicación.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $color

    El color de los íconos (por defecto es quinta).

    		Color— none
    $font-size

    El tamaño de la fuente de los íconos (por defecto es 1.5rem).

    		String— none
